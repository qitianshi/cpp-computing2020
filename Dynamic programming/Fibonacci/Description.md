# Fibonacci
Find the nth Fibonacci number.

## Description
Find the `N`th Fibonacci number, `F`.

## Constraints
`N ≥ 0`, `N ∈ ℤ`

`0 ≤ F ≤ 18 446 744 073 709 551 615` (2×10⁶⁴; `unsigned long long`)

## Input
*From the console:*
* `Line 1`: the integer, `N`.

## Output
*To the console:*
* The `N`th Fibonacci number.

## Sample I/O
### Input 1
```
10
```

### Output 1
```
55
```

---

### Input 2
```
93
```

### Output 2
```
12200160415121876738
```
