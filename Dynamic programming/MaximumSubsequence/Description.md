# Maximum subsequence
Find the largest sum for a contiguous subsequence.

## Description
Given a sequence of numbers, `S`, find the largest possible sum for a contiguous subsequence.

## Constraints
`Sᵢ ∈ ℤ`

## Input
*From `data.txt`:*
* `line 1`: the sequence, `S`.

## Output
*To the console:*
* The sum of numbers in the subsequence.

## Sample I/O
### Input
```
-2 1 -3 4 -1 2 1 -5 4
```

### Output
```
6
```
